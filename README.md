# Data Professional Survey Breakdown Dashboard

## Project Overview
An interactive Power BI dashboard designed to analyze and break down demographic, career, and satisfaction trends among data professionals globally. This project transforms raw survey data into actionable insights, focusing on factors influencing job roles, salaries, tool preferences, and workplace happiness.

## Key Insights & Features
* **Global Participation Tracker:** Uses a geographical treemap to visualize the distribution of survey takers across major hubs like Canada, India, the UK, and the USA.
* **Compensation Analysis:** A horizontal bar chart mapping average salaries across various core roles (Data Scientist, Data Engineer, Data Architect, Data Analyst, etc.).
* **Tech Stack Preferences:** A stacked column chart exploring the popularity of programming languages (Python, R, C/C++, JavaScript, Java) segmented by job titles.
* **Workplace Sentiment Gauges:** Interactive KPI gauges tracking critical modern workplace metrics: **Work-Life Balance** and **Salary Satisfaction**.
* **Difficulty & Sentiment Split:** A donut chart breakdown analyzing user sentiment regarding the overall difficulty of breaking into or navigating the data domain.

## Tech Stack Used
* **Business Intelligence:** Power BI Desktop
* **Data Modeling & Analytics:** DAX (Data Analysis Expressions)
* **Visualization Techniques:** Treemaps, Stacked Column Charts, Gauge Charts, Donut Charts, and KPI Cards.
